## Problem Summary

Educational institutions, hostels, and small businesses are increasingly exposed to cyber threats delivered through malicious links, QR codes, and document files. Existing solutions are often expensive, fragmented, or difficult to manage. This project aims to develop a low-cost, scalable system that detects multi-format cyber threats in real time and provides actionable alerts to administrators.

## Current Direction

Until Harvio Tech Industries confirms detailed expectations, the team is proceeding with a standalone web-based prototype that includes:

- threat scanning for URLs, QR codes, and documents
- a unified threat scoring engine
- an admin dashboard
- actionable alerts and scan history

## Core MVP Features

1. Scan suspicious URLs
2. Decode and inspect QR codes
3. Upload and inspect documents
4. Classify results as `Safe`, `Suspicious`, or `Dangerous`
5. Show clear alerts and history in a dashboard
