# Abstract

Cyber threats targeting educational institutions, hostels, and small businesses are increasing through malicious links, QR codes, and document files. Many existing cybersecurity tools are costly, difficult to manage, or designed to detect only one threat format, making them unsuitable for low-resource environments. This project proposes `CyberShield`, a low-cost and scalable web-based system that detects suspicious URLs, QR-embedded threats, and risky document content in real time. The system analyzes multiple input formats, assigns a threat level, and provides actionable alerts to administrators through a simple dashboard. The proposed solution is intended as a practical prototype for organizations that require accessible cyber protection without complex infrastructure. CyberShield aims to improve early threat detection, reduce exposure to phishing and malware-related attacks, and provide a flexible foundation for future integration into broader security platforms.
